Java语言发展历程：

1996年发布1.0版本; 2004年发布1.5版本从此命名改为5.0,这个版本是自1.1版本来最大的改动，增加了泛型、for each循环、可变元参数、自动装箱、
元数据(即注解)、枚举、静态导入; 2014年发布Java 8，这个版本是近20年最大的改动，提供了lambda表达式等。
